# Portfólio Jonas Augusto

Repositório do meu portfólio pessoal hospedado em [jonasaugustosantos.github.io](https://jonasaugustosantos.github.io).

## Tecnologias Utilizadas

- **HTML5, CSS3, JavaScript (Vanilla)**
- **Tailwind CSS (via CDN)** - Para estilização responsiva e moderna.
- **Integrações via CDN:**
  - **AOS:** Animações ao rolar a página.
  - **GSAP:** Animações avançadas.
  - **Typed.js:** Efeito de digitação no Hero.
  - **EmailJS:** Envio de e-mails pelo formulário de contato.

## Funcionalidades

- **Design Responsivo:** Layout adaptável para desktop e mobile.
- **Menu Hamburguer:** Navegação responsiva para dispositivos móveis.
- **Animações:** Integração com AOS e GSAP para transições e entradas suaves.
- **Formulário de Contato:** Integração com EmailJS para envio direto para meu e-mail.
- **Download de CV:** Link para download do currículo em PDF.
- **Seções Dinâmicas:** Sobre, Experiência (com métricas), Projetos (com stacks), Habilidades, Cursos e Contato.

## Estrutura

- `index.html`: Página única contendo todo o conteúdo, estilos e scripts.
- `jonasaugusto_cv.pdf`: Currículo em formato PDF para download.
- `README.md`: Este arquivo.
